# vant-demo

## Project setup
```
yarn install
或者 cnpm i
```

### Compiles and hot-reloads for development
```
yarn serve
或者 cnpm run serve
```

### Compiles and minifies for production

#### 生产环境
```
yarn build
或者 npm run build
```
#### 测试环境
```
yarn build-test
或者 npm run build-test
```
#### 开发环境
```
yarn build-dev
或者 npm run build-dev
```

###运行本地dis包
```
serve -s dist
```
